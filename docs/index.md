## 理論

今回使用するステッピングモーターの仕様

![](../img/stepper/stepper_1.jpg#center)

Step angle: 1.8°と記載されております。1.8°ステップ角度ということはステッピングモーターを一回転(360°)させるためには

```
360 / 1.8 = 200  
```

200ステップ必要ということになります。
